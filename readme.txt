Inserta fichas de películas, series, programas y documentales en tu blog

== Description ==

Gracias a este plugin, podrás añadir fichas de Series.ly como widgets en tu página web.
Además habilita un botón en el editor tinyMCE de Wordpress para insertar estas fichas
también en tus entradas.

Para un uso avanzado de los shortcodes, usa la siguiente estructura:
[sly-widget q="{nombre de la ficha}" size="{tamaño}" border="{borde}" /]

{Tamaño} puede ser:
 - 0: Estándar
 - 1: Alargado
 - 2: Cuadrado

{Borde} puede ser:
 - 0: Ninguno
 - 1: Borde fino
 - 2: Borde grueso

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `sly_widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
